## Metodologia de Análise – Reconnaissance
Todas as análises deste projeto são feitas sob perspectiva de um analista SOC/Blue Team. <br/>
É um projeto de nível teórico utilizando o framework da MITRE, ATT&CK.

*Esse projeto foi feito com intensão de mapear táticas de reconhecimento e é deliberadamente limitado, servindo melhor apenas como aprendizado e portfólio*

## Por que Reconnaissance foi a tática escolhida?
A fase de reconhecimento é o instante que o atacante não usa força bruta, quer apenas coletar informações dos sistemas. Logo, reconhecer uma varredura suspeita de minima intensidade como sendo de um usuário malicioso, é a chave para impedir um ataque que gere ainda mais prejuizos à empresa. Quanto mais cedo neutralizar a ameaça, sob mais segurança o sistema é mantido.
